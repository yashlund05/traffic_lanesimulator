# Traffic Queue Management System (Data Structures Project)

## Overview
A local web app demonstrating queues (Unit-1) implemented using linked lists (Unit-2).  
Each lane is a queue; enqueue/dequeue operations use a linked-list-backed queue (`LinkedQueue`) and `VehicleNode` as linked nodes.
